How Memory Works

MinouChat uses a 3-layer memory system to personalize your conversations. Each layer has a different purpose and priority level.

1

User Profile (Highest Priority)

Information you explicitly set in "About You" - your name, intro, feedback preferences, topics to avoid.

2

Backstory

The persona's character background. Relevant parts are retrieved based on what you're discussing.

3

Learned Facts

Things the AI learns about you from conversations - your job, hobbies, preferences, goals.

When you chat, all three layers are combined into context that the persona can reference. Higher priority layers are always included first.

Learned Facts

As you chat, MinouChat automatically extracts facts about you. For example, if you mention "I'm a software engineer in Seattle", the system learns:

  • occupation: software engineer
  • location: Seattle
Self-Correcting

If the AI learns something wrong, just correct it naturally in conversation. Say "Actually, I moved to Portland" and the old fact will be updated.

Fact Types

Managing Facts

Visit the Memory & Facts page to:

  • View all facts the AI has learned about you
  • Edit any fact to correct it
  • Delete facts you don't want remembered
  • Filter by persona to see character-specific facts

Best Practices

Building Rich Memory

  • Have natural conversations - mention your name, job, hobbies, and preferences
  • Share context about your life - the more the AI knows, the better it can help
  • Correct mistakes naturally: "Actually, I prefer X" will update the fact

Controlling Memory

  • Visit /memory regularly to review what's been learned
  • Delete any incorrect or unwanted facts
  • Edit facts to be more accurate

Personalizing Personas

  • Fill out the "About You" section for each persona you use frequently
  • This gives the persona explicit context about you (not guessed)
  • Set your preferred feedback style: supportive, direct, or balanced
  • List topics to avoid if there are sensitive subjects

Per-Persona vs Global Facts

Understanding Scope

Global facts apply to all personas - like your name or occupation.
Character-specific facts only apply to that persona - like topics discussed with a particular coach.

Privacy

Privacy-First Design

MinouChat is designed with your privacy in mind. When using Ollama (local), all your conversations and fact extraction happen entirely on your machine. Nothing is sent to external servers.

How Your Data is Stored

  • Conversations: Stored locally in SQLite database
  • Facts: Stored locally, never sent to cloud services
  • Documents: Stored and processed locally

Cloud Providers (Optional)

If you choose to use cloud providers (OpenAI, Anthropic, OpenRouter), your conversations are sent to those services. Check each provider's privacy policy for details.
